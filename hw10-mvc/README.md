# Домашняя работа № 10

- Реализация взаимодействия AJAX с REST апи по вводу каталога книг в библиотеке

# Как запускать:
- в одной консоли запустить само приложение (mvn spring-boot:run)
- во второй - запустить пакет frontend через node. Вот порядок команд (для linux):
  - `cd frontend`
  - `npm install`
  - `npm run dev`
- Приложение будет доступно для взаимодействия на 3000-м порту по умолчанию.
