# Домашняя работа № 13

- Реализация SS ACL для приложения по вводу каталога книг в библиотеке
- Введение трех видов прав в приложении: Наблюдатель, Редактор, Админ
- Наблюдатель может лишь читать данные (видеть все списки книг, авторов, жанров)
- Редактор может читать, добавлять и редактировать данные (книги, авторы, жанры)
- Админ может читать, добавлять, редактировать, удалять данные (книги, авторы, жанры, комментарии)
- Админ также может создавать права
